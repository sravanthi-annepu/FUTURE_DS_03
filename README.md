# 📊 E-Commerce Conversion Performance Analysis(Power BI)

## 🔍 Overview

This project analyzes user behavior in an e-commerce platform using a funnel approach. It tracks how users move from **Visitors → Leads → Customers** and identifies key drop-off points.

---

## 🎯 Objective

* Analyze conversion rates across funnel stages
* Identify drop-offs in user journey
* Evaluate category performance
* Provide actionable business insights

---

## 📊 Dashboard Features

* Conversion Funnel (Visitors → Leads → Customers)
* KPI Cards (Visitors, Leads, Customers, Conversion Rates)
* Conversion Metrics (Traffic to Lead, Lead to Customer)
* Top Product Categories Analysis
* Lead Sources Performance
* Conversion Trend Over Time
* Device Distribution
* Key Recommendations Section

---

## 🛠 Tools Used

* Power BI
* Excel / CSV Dataset

---

## 📈 Key Insights

* Significant drop-off observed between Visitors and Leads
* Strong Lead to Customer conversion (~69%)
* Smartphone category dominates sales performance
* Conversion trends peak during mid-month

---

## 💡 Recommendations

* Improve product page UX to increase add-to-cart rate
* Optimize checkout process
* Focus on high-performing product categories
* Implement retargeting campaigns

---

## 🚀 Outcome

This project demonstrates how data-driven insights can improve conversion rates and business performance in e-commerce platforms.
